Папки: 
css - папка со стилями

Файлы:

404.php - скрипт, выводящий ошибку 404.
com.php - скрипт, позволяющий оставлять комментарии.
index.php - основная часть сайта. 
city.php - пробрасывает на введенный пользователем город.
form_register/auth.php - верстка регистрации/авторизации.
auth/reg.php - скрипты регистрации и авторизации
zd12.sql - база данных, для того, чтобы все работало, импортируй данный файл в свой phpMyAdmin.


Все взаимодействия с бд производятся через команды а-ля  $mysql = new mysqli('localhost','root','','kurs');
Если у тебя другие названия пользователя или есть пароль - конструкцию надо будет соответствующе изменить, например:
	$mysql = new mysqli('localhost','ПОЛЬЗОВАТЕЛЬ','ПАРОЛЬ','НАЗВАНИЕ БД');

Данные по погоде берутся с сайта https://openweathermap.org/. Используется API. Можно использовать API, введенный в скрипте страницы Index по умолчанию, 
но я бы рекомендовал произвести регистрацию и ввести свой API, заодно описав данный процесс.
